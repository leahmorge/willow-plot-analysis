# Willow Field Plot Analysis

## Overview
This project analyzes data collected from multiple willow field plots to explore patterns in vegetation characteristics and environmental conditions. The analysis is conducted primarily in R, with supporting data preparation in Excel.

## Objectives
- Compare willow characteristics across multiple field plots  
- Identify patterns or differences between sampling locations  
- Demonstrate data analysis and visualization using R  

## Tools & Technologies
- R (tidyverse, ggplot2, etc.)
- R Markdown
- Excel (data preparation)

## Data
The dataset used in this project is the property of The Trustees of Reservations and is not publicly available.  
Data for this project was collected during the 2023-2025 field seasons, with contributions from the author as part of field data collection efforts.

## Methods
Data from four field plots was cleaned and organized before being analyzed in R. Multiple R Markdown files are used to explore patterns within and across plots, with visualizations created to support interpretation.
